The Longterm Project

This is a project that is near and dear to my heart. This is what Unrestricted Coding started out as. I had plans to make a whole site dedicated to teaching people how to code. However I now see how much time that is really going to take. So I am modifying Unrestricted Coding to be much more than that. It will be the central hub for all things coding and projects for me.

I still want to make this book, so I will start to build it slowly and over time. It may change as time goes on, adapting to new technologies. Now what this book is, is a single guide to programming. My goal is to make a book that anyone can read, starting with zero programming knowledge. They should be able to read through the whole series laid out below, and by the end of it pick up and build projects of their dreams.

Too many times to people set out to learn programming only to hit paywalls, and non-linear information. I'm hoping to break that stigma with this book.

Community Built

While this is my project, and I look forward to building it and making it great, I understand I don't know everything. I therefore am opening the book up to the community of github to allow contributors and other writers to help out. Maybe if a few people get active with this project my true goals will be achieved. Free and Open information from zero to programmer. If you would like to help, please reach out.
